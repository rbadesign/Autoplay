Версия для автовоспроизведения с переносного носителя

Инструкция по применению:
1. Записать содержимое на компакт-диск или флэш-носитель
2. Вставить носитель в компьютер

Если под Windows разрешён автозапуск содержимого с переносных носителей, то автоматичеси запустится воспроизведение видео.
Список воспроизводимых видео-файлов перечислен в плэй-листе cryo-cell.xspf
